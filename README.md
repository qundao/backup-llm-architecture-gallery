# Backup: LLM Architecture Gallery

by **Sebastian Raschka**

> source: <https://sebastianraschka.com/llm-architecture-gallery/>
>
> Total 65+ LLM model architectures (DeepSeek R1, DeepSeek V3 share same architecture image/).

- Script usage:

```shell
# download latest llm arch images
pip install -r requirements.txt
python src/main.py

# stitch all images
pip install pillow
python src/stitch-imags.py  # --save, --height
```
